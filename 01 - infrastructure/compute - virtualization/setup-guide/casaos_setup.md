# 🛠️ (SOP) Instalando e configurando o CasaOS - Raspberry Pi

### 📝 Descrição e Escopo

O CasaOS não é um sistema operacional completo no sentido estrito, mas sim uma interface de gerenciamento (UI/Dashboard) leve e baseada em ecossistema Docker, rodando tipicamente sobre uma distribuição Linux (como Debian ou Ubuntu) nativa da arquitetura ARM do Raspberry Pi.

A função dele no cenário proposto é rodar alguns serviços essenciais para o meu dia-a-dia em containers, facilitando também a manutenção e operação dos containeres, uma vez que tem uma interface web intuitiva e uma loja com alguns serviços prontos para a instalação e utilização em poucos cliques.

##

### ℹ️ Pré-requisitos

Como ele é uma ferramenta instalada sobre uma distribuição base, é muito importante já estar com uma distribuição Linux devidamente instalada.
